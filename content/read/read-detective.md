# **关于脑洞**

这个书单原本是针对本格推理小说的，但搜罗书籍，阅读的过程中，逐渐接触了许多优秀的非本格作品，还有甚至算不上推理小说的悬疑故事。这些故事优秀到如果不好好分享出来，我一定会后悔的程度。
 
所以我把书单的名字从“推理的宇宙”，修改成了“脑洞的宇宙”，书单虽然以日系为主，但并不限于日本作家。传统意义上的经典作品（如阿加莎，柯南道尔）之类的不再赘述，谨慎罗列，鞠躬致意。

# **关于推理小说**

推理小说起源于欧美，但巅峰几乎是在日本。过去百年，从江户川乱步开始，日系推理百花齐放，各种流派和脑洞齐飞，衍生出了如本格推理，社会派推理等有趣的分支。加上日本文化独有的黑色底色，乱神宗教的背景，这百年间战争及邻国之间令人唏嘘的历史，都让日本成为了一个犯罪和文化交织，罪恶之花很容易盛开的地方。

大概小学的时候，名侦探柯南进入了中国大陆。从那时候我就第一次感觉到脑洞的迷人之处，后来的福尔摩斯，阿加莎等传统节目，自然也在长大的过程中，丰富了自己的视野。严格来说，柯南应该可算日系推理~~不是~~？但最开始接触日系的小说，反而是从白夜行，嫌疑人X，火车这类畅销书开始的。如今比起来，东野圭吾更像是日系推理世界里的肥皂剧导演，高产，故事有趣，偶尔牛逼；但显然，日系推理宇宙水深的很……

# **本格推理**

由岛田庄司提出的新本格派写作七大定律，岛田认为：若以绫辻行人的馆系列做为创作目标，自然就应该将馆系列的复用型守则分析出来；又因为馆系列代表了新本格浪潮，所以他将分析后的复用型守则，称之为〈新本格七则〉[[4]](https://zh.m.wikipedia.org/zh-sg/%E6%9C%AC%E6%A0%BC%E6%B4%BE%E6%8E%A8%E7%90%86%E5%B0%8F%E8%AA%AA#cite_note-4)。

一、事件的舞台必须是封闭空间，例如孤岛或[暴风雪山庄](https://zh.m.wikipedia.org/wiki/%E6%9A%B4%E9%A3%8E%E9%9B%AA%E5%B1%B1%E5%BA%84)。登场人物无法自由出入，也必须排除拥有先进鉴识技术的警方。亦即，必须保证只有逻辑思考才能破案的必然性。

二、屋内的各房间都要能够上锁，属于重视隐私的建筑物。

三、屋宅的主人或客人，全员在小说开头都必须介绍给读者，凶手就在其中。

四、发生了各种事件，并导向血腥的惨剧。

五、侦探登场，是后来才出场，或是一开始就在屋内都可以。

六、惨剧依然继续发生，但凶手的身份依然不明。侦探开始提出推理，读者可以跟着斗智。

七、侦探在故事最后指出凶手，必须是令读者意外的人物。如果无法做到，就不能称为成功的作品。

# **作家、书单和推荐**

目前我仍然在沉迷中，本书单目测仍然会以周为单位进行更新。

## **推理之神 岛田庄司** 

本格推理的核心人物，脑洞极大，文学性同样很强。大时代背景下的故事本身很多时候反而比诡计更加深刻。

- [《占星术杀人魔法》](https://book.douban.com/subject/10740776/) 👍👍
  - 出道即巅峰，占星术是“分尸的岛田”的代表作，代表了文学性和推理性的极佳组合。复盘的时候你会不自觉的想，这诡计在别的书，电影，柯南里，似乎都见过，但这些文字成功的让你一叶障目，最后只能在心里喊一句牛逼合上封底。
- [《异邦骑士》](https://book.douban.com/subject/11501902/)👍👍
  - 你拿着桌子上的剧本发愁到底怎么回事的时候，岛田君掀了桌子告诉你，其实我们玩的是乒乓球，你手里拿的其实是球拍，傻逼了不？
- [《黑暗坡食人树》](https://book.douban.com/subject/23780809/)
  - 故事仍然牛逼而黑暗，虽然离神作似乎远了些，但终究值得一读。
- [《斜屋》](https://book.douban.com/subject/10740772/)
  - 褒贬不一的一本书，喜欢的很喜欢，不喜欢的则觉得偏离本格有些太远，我自己不算太喜欢，但毕竟有人评价极高，希望你喜欢。
- [《北方夕鹤》](https://book.douban.com/subject/20468727/)👍
  - 从诡计的角度，这并不是一部多复杂的推理犯罪，但这故事本身，我很喜欢。
- [《奇想天动》](https://book.douban.com/subject/23780806/)👍
  - 一个诡计跨越了几十年，这本身就是时代之神留下的痕迹。
- [《螺丝人》](https://book.douban.com/subject/25859250/)👍
  - 我个人极其喜欢的一本书，因为你会发现，脑洞是可以降为打击的……
- [《出云传说》](https://book.douban.com/subject/3723863/)
  - 中规中矩的好故事，岛田的功力不减，但跟几本神作来比，确实也差了点意思。
- [《屋顶上的小丑》](https://book.douban.com/subject/30457014/)
  - 这本不是太行……
- [《鸟居密室》](https://book.douban.com/subject/35231665/)
  - 有点像乱步了，因为不是本格故事，所以显得诡计的说服力不够强。
- [《溺水的人鱼》](https://book.douban.com/subject/27197023/)
  - 这不是一本推理故事，所以不在这里推荐了。但这是一个好故事，如果喜欢文学性的东西，值得一读。
- [《晕眩》](https://book.douban.com/subject/24838488/)👍
  - 岛田的舞台，是整个世界。
- [《幻肢》](https://book.douban.com/subject/35797392/)
  - 岛田大哥最新的作品，我觉得呢，有点太过于放飞自我了。



## **百年经典 江户川乱步**

日系鼻祖，用如今的审美来看，很多故事相对格局并不惊人，但这人的不可描述的取向造就的极其阴暗的故事感，甚至奠基了整个日系推理的基调，也是我喜欢日系的一个重要原因。总的来说，乱步的短篇更好看一些，因为很像是极其传统风格的诡计缝合怪，让人佩服不已。

- [《阴兽》](https://book.douban.com/subject/6558049/)👍
  - 如果我说我最喜欢的是这本，你可能会觉得我变态。但……这绝对是乱步的核心竞争力。
- [《D坂杀人事件》](https://book.douban.com/subject/6060436/)👍
  - 有趣的故事，有趣的诡计。你很难想象这是百年前人的脑洞。
- [《人间椅子》](https://book.douban.com/subject/6558046/)
  - 很多人把这本书推荐为乱步的入门作品，但实际上椅子更像是中规中矩的柯南TV剧，而不是剧场版。
- [《两分铜币》](https://book.douban.com/subject/6001119/)
  - 这是我的乱步第一本书，倘若读在后来的西泽之后，一定不会有当时那么深的佩服感了，但乱步终究是乱步，还是值得翻一翻的。
- [《魔术师》](https://book.douban.com/subject/10801045/)
- [《孤岛之鬼》](https://book.douban.com/subject/6967992/)
  - 两部长篇怎么说呢，在当时必然是经典，在如今这个脑洞盛开的日系宇宙里，的确显得单薄了些。

## **绑架的 冈岛二人**

分尸的岛田，绑架的刚岛，太多书籍都没有中文版能买，克莱因壶之外似乎都不够惊艳。国内能找到的这几本都不错，但首推克莱因壶。

- [《克莱因壶》](https://book.douban.com/subject/33658616/)👍👍
  - 在盗梦空间前如此多年，这样一个脑洞里的脑洞的故事，早已构思完整。即使你看过盗梦空间多遍之后，自觉对这诡计熟悉无比，这个故事仍然能够给你新鲜的神经触觉。
- [《99%诱拐》](https://book.douban.com/subject/35217275/)👍
  - 几乎刚开始就告诉了你故事的全貌，但你仍然无法摸到真相。个人非常喜欢这个故事。
- [《然后，门被关上了》](https://book.douban.com/subject/34998166/)👍
  - 唯一的缺点，是人设真的很怪。但故事和诡计足够引人入胜，似乎也就足够了。

## **新本格的开端 绫辻行人**

号称岛田之后最强本格！怎么说呢，馆系列甚至是馆宇宙，的确可以代表一位大师深厚无比的功底。但暗黑馆作为整个馆系列的前传和大总结，居然走上了架空世界的路，的确让我这种传统人士很难接受，也有些偏离了本格的初衷。而深受好评的《替身》系列，则直接开始放飞自我了……如果你沉迷传统推理，那么馆系列则是绫辻的必读作品，也是唯一需要读的作品。

何况，失忆太多是我后来越来越不太欣赏绫辻的一个关键原因。

- [《十角馆事件》](https://book.douban.com/subject/26771717/)👍
  - 想到水母那本书，核心的不在场证明异曲同工，作为馆系列的第一部也是出道作品，同时也宣告了绫辻从不破案，只是告诉你[或然故事](/2022/01/17/160.html)的推理风格。
- [《钟表馆事件》](https://book.douban.com/subject/26771719/)👍👍
  - 最喜欢的一部作品，悲伤的故事，宏大的诡计，但也是从这一本开始，奠定了这货“失忆的绫辻”的称号~~不是只是我这么叫他的~~。
- [《水车馆事件》](https://book.douban.com/subject/26771723/)
  - 一个很容易的诡计，一个好故事。
- [《迷宫馆事件》](https://book.douban.com/subject/26771720/)👍
  - “有的密室，是为了创造这里没有密室的假象。”
- [《人偶馆事件》](https://book.douban.com/subject/26771724/)
  - 很不喜欢这本的叙事风格，太讨巧了。
- [《黑猫馆事件》](https://book.douban.com/subject/26771722/)👍
  - 有些联想到岛田的《晕眩》，显然，看过晕眩以后，解决好猜了很多。
- [《暗黑馆事件》](https://book.douban.com/subject/26771721/)👍
  - 如果你喜欢馆系列，那么这一本冗长而华丽的大故事值得一读。如果你只喜欢硬核的推理本，那么这一本也许会让你失望。
- [《惊吓馆事件》](https://book.douban.com/subject/26771718/)
  - 你看，失忆容易起来了。
- [《奇面馆事件》](https://book.douban.com/subject/26771726/)
  - 岛田洁亲自下场的故事，总以为能更厉害一些。
- [《魔女的诅咒》](https://book.douban.com/subject/35341348/)
  - 铺垫越深，尾声就会越失望，很有一种我裤子都脱了，你就给我看个这的失望感。失忆真是万能药啊……
- [《雾越邸暴雪谜案》](https://book.douban.com/subject/27020856/)👍
  - 馆系列之外这是读到唯一一本还可以的，但仍然因为少了硬核的宏大故事，让动机，诡计，都显得苍白很多。
- [《小丑的追魂曲》](https://book.douban.com/subject/35216487/)
  - 很难不想到《奇想，天动》，但显然，格局因为强行诡计而小了太多。


## **严肃推理 松本清张**

社会派本质上应该属于严肃文学的一个分支才对，严格来说，逻辑感这方面确实不是特长。松本清张的故事很真实，很平，往往很沉重。脑洞不深，但背后的心情不愉悦，读完以后没有逻辑圆满的快感，很难受。倘若我是在念大学那会接触到，大概会更对胃口一些。

- [《水之烟》](https://book.douban.com/subject/26803473/)
- [《黑色皮革手册》](https://book.douban.com/subject/27193113/) 👍
  - 一定要推荐的话，推荐这本。
- [《万分之一的偶然》](https://book.douban.com/subject/19954848/)
- [《一个背叛日本的日本人》](https://book.douban.com/subject/6848987/)👍
  - 非常喜欢这个故事啊。
- [《眼之壁》](https://book.douban.com/subject/30195105/)

## **朴素而惊人的 西泽保彦**

推理故事最漂亮的模样，大概就是如此了，没有炫技，故事，脑洞，扎实的人设，轻轻松松的讲给你听，如果说最崇拜的推理作家是岛田，那最喜欢的，必须是西泽。

首先是匠千晓系列，微信读书收录不全而且顺序收录不正确，有点难受，建议按照下面这个顺序阅读。

- [《解体诸因》](https://book.douban.com/subject/27078155/)👍👍
  - 最好的推理故事，大约就是这个模样了，西泽的神作之一。
- [《她死去的那一夜》](https://book.douban.com/subject/26373943/)👍👍
  - 这诡计最关键的地方，在最开始就告诉了你。你猜测了所有可能性，但终究低估了人性。
- [《啤酒之家的冒险》](https://book.douban.com/subject/20559358/)
  - 这本是微信读书唯一没有的一本，先列在这……
- [《羔羊们的平安夜》](https://book.douban.com/subject/26941499/)
  - 是个好故事，但动机稍微……诡异了些。
- [《苏格兰游戏》](https://book.douban.com/subject/26421649/)👍
  - 仍然是动机有些诡异的一本书，但就凭女主的人设也得推荐啊。
- [《依存》](https://book.douban.com/subject/26383897/)👍
  - 如果这个系列是一套美剧，这一集就是坑挖的最深的那一集。
- [《黑贵妇》](https://book.douban.com/subject/27109948/)
  - 中规中矩的故事，丰富了这个系列丰满的人设。
- [《替身》](https://book.douban.com/subject/27200982/)
  - 你从文本的多时间线，就能猜测到诡计的核心。但铺垫的所有背景故事一个一个的串起来，还是非常带感。
- [《怜悯恶魔》](https://book.douban.com/subject/35058743/)👍
  - 比起诡计，这个世界的因果往往不是非此即彼的。比起诡计，这是一个动机和情感交织的好故事。

然后是单本，总体来说也值得一读，但跟千晓系列比，有点工业啤酒和手工精酿的意思————就是差点意思的意思。

- [《死了七次的男人》](https://book.douban.com/subject/27055125/)👍
  - 想到一个玩过的剧本杀……于是总是有点出戏。
- [《完美无缺的名侦探》](https://book.douban.com/subject/19951341/)
  - 怎么说呢，有点没有get到核心设定……所以侦探其实是个主线故事里的，外挂？

## **黑 乙一**

黑乙一，毫无底线的恐怖故事和匪夷所思的脑洞，他们是最原始的世界里最冷漠的一面。如果你也是个孤独的人，应该会理解这些故事存在的意义。

- [《动物园》](https://book.douban.com/subject/26723423/)👍
  - 最经典的黑色短篇集合，不可错过。
- [《只有你听到》](https://book.douban.com/subject/35216488/)👍
  - 有时候，想成为这样的作家，因为他们的脑洞很现实，又很温柔。
